Eric's Recipe Box: Interactive Recipe Finder — Site Plan

A concise planning document outlining name, purpose, scenarios, color schema, typography, and wireframes.

Site name

Name: Eric's Recipe Box

Reason: The name feels friendly and easygoing—just like the site! It’s all about helping beginners find simple, fun recipes and giving a little nudge with tools like the random recipe picker and “What Can I Bake?” helper, so no one ever feels lost or stuck.

Site purpose

Purpose: Provide a beginner-friendly baking hub with practical, step-by-step content and two dynamic features: a Random Recipe Generator powered by a public recipe API, and a “What Can I Bake?” tool that suggests recipes from a JSON dataset based on user-provided ingredients.

  • Information: Starter tips, essential tools, and simple baking guidance.
  • Dynamic features: Random dessert recipes via API; ingredient-based suggestions via local JSON.
  • Design goals: Valid HTML/CSS, accessibility, responsive layout, and clear UX.

Scenarios

  • Scenario 1: “I have flour, butter, and chocolate chips—what can I bake tonight?”The site suggests matching recipes using the JSON-based ingredient filter.
  • Scenario 2: “I want to try something new—surprise me.” The site fetches anddisplays a random dessert recipe (title, image, instructions) from a public API.

Color schema

Seal Brown #63310D Headings, primary accents, buttons
Warm Cream #FFF8E6 Page background, cards
Charcoal Text #2F2F2F Body text for readable contrast

Typography

  • Fredericka the Great (700): Applied to major headings (h1–h2) for a friendly, classic tone.
  • Roboto (400, 600): Applied to body text, lists, and labels for clean readability and accessible line height.

Accessibility and UX targets

  • Contrast: Dark charcoal text on warm cream background meets readability targets.
  • Structure: Semantic headings and landmark regions (header, main, footer).
  • Controls: Clear buttons/links with focus styles and descriptive labels.
  • Responsive: Readable on mobile-first, scales to desktop.
